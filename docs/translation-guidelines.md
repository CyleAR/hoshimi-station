# 번역 주의사항

로그인창이나 공지 영역에 붙일 임시 안내문입니다. 필요하면 이 문서를 기준으로 문구를 다듬어 주세요.

## 기본 원칙

- 원문의 의미를 유지하되 한국어로 자연스럽게 읽히게 번역합니다.
- 캐릭터 말투, 호칭, 고유명사는 기존 번역과 최대한 맞춥니다.
- 같은 용어가 여러 곳에 나오면 같은 번역을 사용합니다.

## placeholder

- `{user}` 같은 placeholder는 지우거나 번역하지 말고 그대로 둡니다.
- 원문에 있는 placeholder가 번역문에도 들어가야 합니다.
- 순서가 자연스럽지 않으면 문장 안에서 위치만 조정합니다.

## 줄바꿈과 기호

- 원문에 `\n`이 있으면 번역문에도 필요한 위치에 `\n`을 유지합니다.
- `♪`, `♡`, `!`, `?` 같은 감정 표현 기호는 문맥에 맞게 유지합니다.
- 너무 긴 한 줄은 UI에서 읽기 힘들 수 있으니 자연스러운 위치에서 줄바꿈합니다.

## ADV 이름/화자

- ADV의 `name` 항목도 번역 대상입니다.
- `{user}`는 매니저/플레이어 placeholder이므로 그대로 둡니다.
- 캐릭터 이름은 기존 표기와 맞춥니다.

## 저장 전 확인

- 오타, 누락된 placeholder, 과한 의역을 확인합니다.
- 이미 번역된 문장을 수정할 때는 주변 문맥과 연결 항목을 같이 확인합니다.
